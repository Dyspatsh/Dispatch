About Dispatch

What is Dispatch?

Dispatch is a Tor-only, privacy-focused file sharing and communication platform where users can send encrypted files, engage in private and group chats, and manage their digital presence with complete anonymity. All data is encrypted client-side, and even the platform owner cannot access user content.

Built for privacy advocates, journalists, activists, and anyone who values their digital freedom, Dispatch operates exclusively on the Tor network, ensuring no IP addresses are ever logged or exposed.

How It Works

  • Client-Side Encryption: Files are encrypted in your browser using AES-256-GCM before upload. The encryption key never leaves your device.
  • File Transfer Flow: Send a file → Recipient accepts → File becomes available for download → Auto-deletes after expiry (24-72 hours pending, 7 days after acceptance).
  • Private Chat: End-to-end encrypted messages with read receipts, typing indicators, and message search for Pro+ users. Messages auto-delete after 24 hours.
  • Group Chats: Create groups with unlimited members (Premium only), manage members, promote admins, and enjoy the same encryption and auto-delete features.
  • Account Recovery: Each account is created with a unique 64-character recovery phrase. This is the only way to recover your account if you forget your password or PIN.

Security Features

End-to-End Encryption

All files and messages are encrypted client-side using AES-256-GCM, the same standard used by governments and militaries worldwide. Your data is encrypted before it ever leaves your device, and decrypted only when you choose to download or view it.

Tor-Only Access

Dispatch is exclusively available as a Tor hidden service. Your IP address is never logged, stored, or even visible to the server. Combined with Tor's onion routing, your identity remains completely anonymous.

Two-Factor Authentication

Secure your account with TOTP-based 2FA (Google Authenticator, Authy, etc.). Generate backup codes for emergency access. Each backup code can be used only once.

Account Lockout Protection

After 5 failed login attempts, accounts are locked for 30 minutes. This prevents brute-force attacks while maintaining usability.

Argon2 Password Hashing

All passwords, PINs, and recovery phrases are hashed using Argon2id, the winner of the Password Hashing Competition and the most secure password hashing algorithm available.

XSS Protection

All user-generated content is HTML-escaped before display, preventing cross-site scripting attacks. No user input is ever rendered as raw HTML.

HTTP-Only Cookies

Session cookies are set with HttpOnly and SameSite=Lax flags, preventing JavaScript access and CSRF attacks.

No IP Logging

By operating exclusively on Tor, Dispatch never sees your real IP address. No logs are kept, no analytics are tracked, and no third-party services are used.

File Features

  • Drag & Drop Upload: Simply drag files into the drop zone or click to browse. Upload multiple files at once with queue management.
  • File Preview: See thumbnails for images and file-type icons for documents, videos, and archives before sending.
  • File Encryption: All files are encrypted with AES-256-GCM before upload. The encryption key is generated in your browser and never transmitted.
  • Custom Expiration: Pro and Premium users can set custom expiration times for their files (1-7 days). Free users have standard 7-day expiration.
  • Password Protection: Pro and Premium users can add password protection to files. Recipients must enter the password to download.
  • File Acceptance Workflow: Recipients must explicitly accept files before they can be downloaded. This prevents unwanted content and gives control to the recipient.
  • File History: View all files you've sent and received, with status tracking (pending, accepted, declined, downloaded, expired).

Chat Features

  • Private Encrypted Chat: End-to-end encrypted messages with real-time delivery via WebSocket.
  • Read Receipts: See when your messages are sent, delivered, and read (Premium users can disable).
  • Typing Indicators: Real-time indication when the other user is typing.
  • Message Search: Pro+ users can search their message history by keyword.
  • Message Reactions: React to messages with [Like], [Thanks], [Agree], or [Helpful].
  • Group Chats: Premium users can create groups with unlimited members. Group admins can promote members, remove users, and edit group descriptions.
  • Group Member Management: View all members, see their roles (Owner, Admin, Member), and manage permissions.
  • Message Auto-Delete: All chat messages (private and group) automatically delete 24 hours after being sent.

Account Features

  • User Profiles: View user profiles by clicking usernames in chat. See join date, role, bio, and file statistics.
  • Bio: Pro and Premium users can set a short bio about themselves (max 200 characters).
  • File Statistics: View your total files sent/received, storage used, and active files count.
  • Theme Selection: Choose between light and dark mode. Preference is saved to your account.
  • Login History: View your last 30 days of login activity.
  • Blocked Users: Block users to prevent them from contacting you or sending files.
  • Account Recovery: Recover your account using your unique 64-character recovery phrase (provided at registration).

Admin Features

  • Dashboard Analytics: View real-time charts for user growth, file activity, and storage usage trends.
  • System Health Monitoring: Monitor CPU, memory, disk usage, and server uptime.
  • User Activity Heatmap: Visual representation of user activity over time.
  • User Management: Search, filter, ban, unban, change roles, and delete users.
  • File Management: Search, filter, and delete any file (for content moderation).
  • Batch Actions: Select multiple users for bulk ban, unban, role change, or deletion.
  • Data Export: Export user and file data to CSV format for analysis or backup.
  • Failed Login Monitoring: Track accounts with repeated failed login attempts.

Data Retention & Privacy

  • Files: Automatically deleted after expiration (7 days for free users, custom for Pro/Premium).
  • Chat Messages: Automatically deleted 24 hours after sending.
  • Login History: Stored for 30 days only, then automatically purged.
  • Sessions: Active sessions expire after 7 days of inactivity.
  • No IP Logs: The server never sees or logs your IP address due to Tor operation.
  • No Third Parties: Dispatch does not use Google Analytics, CDNs, or any external services that could track you.

Technology Stack

  • Backend: FastAPI (Python) with SQLAlchemy ORM
  • Database: PostgreSQL
  • Encryption: AES-256-GCM (client-side), Argon2id (password hashing)
  • Real-time Communication: WebSockets
  • Authentication: HTTP-only cookies with 2FA support
  • Frontend: Vanilla JavaScript, HTML5, CSS3 with glassmorphism design
  • Network: Tor hidden service (.onion only)
